# My nvim configuration




### text object

| :   |                 |     |
| --- | --------------- | --- |
| i/a | inside/around   |     |
| ciw | search          |     |

| \r  | rename          |     |
| gd  | goto definition |     |
| gr  |                 |     |
|     |                 |     |

### Navigation

| key | motion               |
| --- | -------------------- |
|     |                      |
| {   | Previous Paragraph   |
| }   | Next Paragraph       |
| [[  | previous highlight   |
| ]]  | next highlight       |
| ]f  | next function        |
| [f  | previous function    |
|     |                      |
| [c  | previous change      |
| ]c  | next change          |
| [d  | previous diagnostics |
| ]d  | next diagnostics     |
| [q  | previous quickfix    |
| ]q  | next quickfic        |
|     |                      |
| gd  | goto definition      |
| gr  | find references      |
| \q  | find symbols         |


## Workflow

|                    |                      |
| ------------------ | -------------------- |
| < space >< space > | list last open files |
| < space > fg       | list git files       |
| < space > z        | find directory       |
